# Example: Search for Upper Pines

## Command
```
/recreation.gov Upper Pines
```

## What Happens

1. **Navigate**: Opens recreation.gov homepage
2. **Search**: Enters "Upper Pines" into the search field
3. **Execute**: Clicks SEARCH button
4. **Results Load**: Displays campground listings and map
5. **Recording Active**: GIF begins capturing your interactions
